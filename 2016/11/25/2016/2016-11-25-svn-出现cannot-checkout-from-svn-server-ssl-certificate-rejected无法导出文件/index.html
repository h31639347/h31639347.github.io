发表于 2016-11-25 | 分类于 工具 | | 阅读次数

之前用ubuntu有遇到过一次，还好还记的解决的方法。方法很简单，删除之前的用户授权信息，在重新授权登录就可以了。博客换了空间，看到好多人还是搜索这个解决文章，所以在重新编写一次，希望可以帮助后来的人。

注意:

该方法使用前提是万不得已的情况下请勿使用，因为删除.subversion文件后，之前保留的授权用户信息将会被删除。意味着你要重新再输入一次账号和密码授权。(对我来说无所谓啦)，同时也不能保证能百分百解决您的问题，如果无法解决还请多百度了，我只是把我遇到几次都是这样解决写下记录。
